# KMBA菁英計畫 2026 — 智慧客服

KT&G 大韓菸草 **KMBA菁英計畫** 官方 FAQ 智慧客服（靜態版）。  
店家可輸入問題，即時取得活動任務、抽獎券、超商禮券、區域競賽等說明。

**線上版本：** [https://zxcvaden-hub.github.io/KMBA-FAQ/](https://zxcvaden-hub.github.io/KMBA-FAQ/)

---

## 功能特色

- **智能語意匹配**：支援錯字、少打字、同義說法（非單純關鍵字比對）
- **雙語回覆**：中文提問 → 繁體中文；英文提問 → English
- **超商禮券／抽獎券分離**：避免「禮券」被誤判為「抽獎券」
- **快捷按鈕**：日常任務說明、抽獎規則、超商禮券說明、拜訪任務說明
- **安全護欄**：菸品售價、法規等問題引導聯繫區域業務
- **純靜態部署**：GitHub Pages 即可運作，無需後端伺服器

---

| 檔案 | 用途 |

## 活動規格摘要（V1.0）

**活動期間：** 2026 年 8 月～2027 年 1 月

### 日常任務（每月上限 1,000 分）

| 任務 | 積分 |
|------|------|
| 客人推薦紀錄 | 250 分 |
| 新品陳列照 | 250 分 |
| 新品 QA 挑戰 | 250 分 |
| 業務 QA 挑戰 | 250 分 |

**積分兌換抽獎券：** 500 分→1 張｜750 分→2 張｜1,000 分→3 張

### 拜訪任務

拜訪簽約店 → 陳列架合照 → SurveyCake 上傳 → 每月審核通過後發券  
每拜訪 1 間店 = 1 張抽獎券（每月上限 5 張）

### 每月排行超商禮券

| 排名 | 禮券 |
|------|------|
| TOP 1–10 | 新台幣 1,000 元 |
| TOP 11–30 | 新台幣 500 元 |
| TOP 31–100 | 新台幣 200 元 |

### 雙月抽獎

- 8–9 月累積 → 10 月抽｜10–12 月累積 → 2027/1 抽（抽完歸零）
- 每次 15 位店家，每位 5,000 元超商禮券

## 本機預覽

直接用瀏覽器開啟 `index.html` 即可（需與 `kb-engine.js`、`kmba-logo.png` 同目錄）。

### 本機伺服器版（選用）

```bash
python server.py
```

瀏覽器開啟：`http://127.0.0.1:8788/`

---

## GitHub Pages 部署

1. 將以下檔案 push 至 `main` 分支：
   - `index.html`
   - `kb-engine.js`
   - `kmba-logo.png`
2. Repo → **Settings** → **Pages**
3. Source 選 **Deploy from a branch** → Branch: `main` → Folder: `/ (root)`
4. 等待 1～5 分鐘後重新整理頁面

---

## 更新 FAQ 內容

1. 編輯 `kb-engine.js` 中的 `ANSWERS` 與 `INTENTS`
2. 上傳至 GitHub
3. Pages 自動更新（約 1～2 分鐘）

同步更新知識庫文件：`KMBA_CLUB_2026_MASTER_KNOWLEDGE_BASE_V1.0.md`
